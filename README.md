# swstack-web — 사회복지 실천 AI 웹앱

> 사회복지사를 위한 대화형 AI 업무 도우미 — 브라우저에서 바로 사용

Claude Code를 몰라도 됩니다. 브라우저에서 접속하고, 무료 API 키만 입력하면 바로 사용할 수 있습니다.

## 시작하기 (3분)

### 1. API 키 발급 (무료)

Google Gemini API를 추천합니다 — **완전 무료**, 일 1,500회 사용 가능.

1. https://aistudio.google.com/apikey 접속
2. 구글 계정으로 로그인
3. "API 키 만들기" 클릭
4. 생성된 키 복사 (AIza로 시작)

### 2. 웹앱 사용

1. 웹앱 접속
2. 왼쪽 하단 **API 설정** 클릭
3. **Google Gemini** 선택 (무료)
4. 발급받은 API 키 붙여넣기 → 저장
5. 원하는 스킬 선택 → 대화 시작!

## 지원 AI 모델

| 모델 | 비용 | 특징 |
|------|------|------|
| **Google Gemini 2.5 Flash** | 무료 (일 1,500회) | 추천 — 빠르고 성능 좋음 |
| ChatGPT (gpt-4o-mini) | 저렴 | OpenAI 플랫폼 |
| Claude (Sonnet 4.6) | 유료 | 최고 품질 |

API 키는 **내 브라우저에만** 저장됩니다. 서버에 전송되지 않습니다.

## 스킬 목록 (22개)

### 사례관리
- **접수상담** — 초기 접수, 긴급도 분류, 동의서 체크
- **욕구사정** — 6개 영역 사정, 생태도, 강점사정, PHQ-9/GAD-7
- **개입계획** — SMART 목표, 서비스 연계 계획
- **상담기록** — 회기별 프로세스 노트
- **사례회의** — 안건 구성, 회의록
- **평가** — 목표달성척도(GAS), 사전-사후 비교
- **종결** — 종결 요약서, 사후관리 계획

### 회계
- **예산편성** — 관-항-목 예산표, 산출근거
- **지출관리** — 품의서, 증빙 체크리스트, 정산서
- **감사대비** — 감사 체크리스트, 증빙 누락 확인

### 홍보
- **홍보콘텐츠** — SNS 게시글, 카드뉴스, 보도자료
- **대외보고서** — 후원자 보고서, 활동보고서
- **행사기획** — 기획안, 체크리스트, 사후보고서

### 행정/보고
- **정부보고서** — 행복e음 실적, 보조금 정산
- **제안서** — 공모사업 제안서 (논리모델, 성과지표)
- **회의록** — 회의록, 결정사항 추적

### 슈퍼비전
- **슈퍼비전** — 사례 슈퍼비전, 윤리 딜레마 분석
- **사업회고** — KPT 월간/분기 회고

### 유틸리티
- **개인정보보호** — 개인정보 마스킹, 동의서 점검
- **법령검토** — 의무신고 판단, 관련 법령
- **자원관리** — 지역사회 자원 DB
- **양식관리** — 기관 양식 생성

```
    ...
```

```bash
npm run dev
# http://localhost:3000 접속
```
